# NotchFlow

[![CI](https://github.com/zzfadi/ios-first/actions/workflows/ci.yml/badge.svg)](https://github.com/zzfadi/ios-first/actions/workflows/ci.yml)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](LICENSE)

A macOS notch app with three integrated developer-focused mini-apps.

See [NotchFlow/README.md](NotchFlow/README.md) for full documentation.

## Quick Start

```bash
cd NotchFlow
open NotchFlow.xcodeproj
```

Or build with SPM:
```bash
cd NotchFlow
swift build
```

## Mini-Apps

1. **Fog Note** - Quick capture and note management
2. **Worktree GUI** - Git worktree discovery and management
3. **AI Config GUI** - Find/manage AGENTS.md, prompts, MCPs, skills

## Requirements

- macOS 14.0+ (Sonoma)
- Xcode 15.0+ or Swift 5.9+
